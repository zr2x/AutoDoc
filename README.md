Stack: Switft, UIkit, MVVM, Combine, CompositionalLayout, DiffableDataSource, NSCache Async/Await, REST API

No storyboard/xib, 3rd party libraries

Implemented:
- News feed with CompositionalLayout and DiffableDataSource
- Page-by-page pagination
- Caching API responses and downloaded images via NSCache
- async/await network layer
- Support for two types of iPad orientation
- Clickable newsItem
